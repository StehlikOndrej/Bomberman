# Bomberman
## Popis
2D klasická hra, kde nesmíte dovolit svému soupeři, aby vás zabil svou bombou dříve než vy ho, tou svojí. Sbírejte vylepšení, klaďte bomby a ničte vše, co kolem sebe uvidíte. První battle royal na světě.
